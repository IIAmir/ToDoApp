<h1 align="center">Clean Architecture ToDoApp</h1>

<h4 align="center">ToDoApplication With Jetpack Compose (MVVM)</h4>

## Features
* 100% Kotlin
* MVVM architecture
* Dependency Injection
* Lifecycle ViewModel
* ROOM Database
* DataStore Preferences
* Compose Navigation
* Accompanist Navigation Animation
* System UI Controller
